Word Importance Explorer: Use TF-IDF on 5 documents and identify top keywords with explanation.

Documents
1. Artificial intelligence is transforming the world.
2. Machine learning is a part of artificial intelligence.
3. Deep learning improves machine learning models.
4. AI is widely used in healthcare and finance.
5. Data science and AI are closely related fields.


Program:

from sklearn.feature_extraction.text import TfidfVectorizer

# Documents
documents = [
    "Artificial intelligence is transforming the world",
    "Machine learning is a part of artificial intelligence",
    "Deep learning improves machine learning models",
    "AI is widely used in healthcare and finance",
    "Data science and AI are closely related fields"
]

# Create TF-IDF model
vectorizer = TfidfVectorizer(stop_words='english')

# Fit and transform data
tfidf_matrix = vectorizer.fit_transform(documents)

# Get feature names (words)
feature_names = vectorizer.get_feature_names_out()

# Convert to array
tfidf_array = tfidf_matrix.toarray()

# Print top keywords for each document
for i, doc in enumerate(tfidf_array):
    print(f"\nDocument {i+1} Top Keywords:")
    top_indices = doc.argsort()[-3:][::-1]
    for index in top_indices:
        print(feature_names[index], ":", round(doc[index], 3))
