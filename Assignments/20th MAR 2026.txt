Types of Problems:

1. Slang / Abbreviations
Examples: u → you, plz → please, asap → as soon as possible

2. Emojis:
Examples: 😂, 🔥, 👍
These may need to be removed or converted to meaning (e.g., 😂 → happy)

3. Typos / Spelling Errors
Examples: amazng → amazing, wat → what

4. Repeated Characters
Examples: sooo → so, okkk → ok

5. Extra Punctuation
Examples: !!!, ???


Preprocessing Steps:

To clean the data before using it in ML:

1.Convert to lowercase
"Hello" → "hello"

2.Remove or replace emojis
😂 → "happy"

3.Expand slang/short forms
"u" → "you", "tmrw" → "tomorrow"

4.Correct spelling mistakes
"amazng" → "amazing"

5.Remove repeated characters
"soooo" → "so"

6.Remove punctuation
"!!!" → ""

7.Tokenization (split words)
"I love ML" → ["I", "love", "ML"]
